# Supabase — Design System

## Identidade
- Fundo: branco puro com único acento emerald (#3ecf8e)
- "Everything else is a calibrated grey ladder"
- Tipografia: Circular (ou Inter) weight 500 para headlines
- Botões: 6px radius (quadrado, nunca pill), texto dark no fundo emerald
- Sem gradientes ou efeitos atmosféricos

## Paleta
- Canvas: #ffffff
- Canvas night: #1c1c1c (code blocks e pricing featured)
- Accent: #3ecf8e (emerald — só CTAs)
- Grey ladder para superfícies
- Purple, yellow, pink só em charts e logos de integração

## Princípios
- Minimalismo técnico extremo
- Profundidade via mockups de produto (dashboards, SQL editors)
- White space = o design
- Credibilidade técnica sem ruído visual

## Aplicação no ITBI Dashboard
✅ Ótimo para seção de tabela de dados
✅ Charts coloridos contrastam bem com base neutra
⚠️ Muito minimalista — pode parecer vazio sem mockups ricos
