# Apple — Design System

## Filosofia
"UI recede so the product can speak" — interface como galeria, produto em primeiro lugar.
Sem gradientes decorativos, sem sombras no chrome, só 1 sombra: nos renders de produto.

## Paleta
- Action Blue: #0066cc (único acento — todos os clicáveis)
- Focus Blue: #0071e3
- Sky Link (dark): #2997ff
- Canvas white: #ffffff
- Parchment: #f5f5f7 (variação rítmica em tiles)
- Dark tile: #272729 / #2a2a2c / #252527
- Nav bar: #000000
- Ink: #1d1d1f
- Muted: #7a7a7a

## Tipografia
- SF Pro Display (≥19px) + SF Pro Text (body) → substituir por Plus Jakarta Sans
- Hero: 56px / weight 600 / tracking -0.28px
- Tile: 40px / weight 600
- Body: 17px / weight 400 / tracking -0.374px / line-height 1.47
- Nav: 12px / weight 400

## Espaçamento
- Base: 8px
- Seções: 80px vertical padding
- Cards: 24px interno, 20-24px gutters

## Sombra
- UMA sombra apenas: rgba(0,0,0,0.22) 3px 5px 30px 0 → só em renders de produto

## Botões
- Primary: pill (#0066cc background, branco)
- Secondary: ghost pill (border Action Blue)
- Radius: 9999px (pills) ou 8px (utilitário)

## Aplicação no ITBI Dashboard
✅✅✅ PERFEITO para base clean — espaçamento e tipografia
✅ Tiles alternando branco/parchment/dark = seções do dashboard
✅ Sem gradientes = dados respiram
✅ 17px body = legibilidade premium em tabelas
