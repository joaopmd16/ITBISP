# Vercel — Design System

## Identidade
- Fundo: near-white (#fafafa) + polarity flip para ink (#171717)
- CTA: ink/black puro — "black is the conversion target"
- Tipografia: Geist (geometric sans) + Geist Mono para técnico
- Botões: pill 100px (marketing) ou 6px (nav) — dois tamanhos coexistem
- Gradiente mesh: 3 pares (blue-teal, violet-pink, coral-amber) — só em heroes

## Paleta
- Canvas: #ffffff
- Canvas soft: #fafafa (background padrão)
- Primary: #171717 (ink)
- Link: #0070f3
- Gradiente develop: #007cf0 → #00dfd8
- Gradiente preview: #7928ca → #ff0080
- Gradiente ship: #ff4d4d → #f9cb28

## Sombras
- Stacked shadows (múltiplos offsets) — nunca sombra única pesada
- 5 níveis de elevação

## Tipografia
- Geist: peso 400–600, tracking negativo obrigatório
- Geist Mono: só para código e labels técnicos
- Sem uppercase, sem peso 700+

## Aplicação no ITBI Dashboard
✅ Muito clean e developer-friendly
✅ Stacked shadows excelentes para KPI cards
⚠️ Identidade muito Vercel/tech — menos fintech imobiliário
⚠️ Gradiente mesh pode parecer genérico fora do contexto
