## Sugestão: Revolut + Linear híbrido

Combinar:
- **Dark mode**: Linear (surface ladder, lavender accent)
- **Light mode**: Revolut (alternância black/white, cobalt violet)
- **Dados/tabelas**: Stripe (tnum, tabular figures)
- **Espaçamento**: Wise (24px radius, base 4px, rich semantic colors)

## Fontes substitutas (sem proprietárias)
- Aeonik Pro / Wise Sans / Linear Display → **Plus Jakarta Sans** (700–800)
- Sohne / Circular → **Inter** (400–600)
- Geist Mono / Linear Mono → **JetBrains Mono**
