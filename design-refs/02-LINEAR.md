# Linear — Design System

## Identidade
- Fundo: near-black (#010102) — dark mode only
- Accent: lavender-blue (#5e6ad2), hover #828fff
- Superfícies: 4 níveis (surface-1 a surface-4)
- Tipografia: Linear Display (headlines) + Linear Text (body)
- Bordas: hairline #23252a → #3e3e44
- Sem sombras — usa ladder de superfícies no lugar

## Paleta
- Canvas: #010102
- Surface-1 a 4: tons escuros crescentes
- Ink: #f7f8f8 (primary), #d0d6e0, #8a8f98, #62666d
- Accent: #5e6ad2

## Border Radius
- 4px chips, 8px botões/inputs, 12px cards, 16px screenshots

## Princípios
- Um único acento cromático (lavender)
- Sem gradientes atmosféricos
- Profundidade via surface ladder, não sombras
- Produto UI screenshots como elemento decorativo principal

## Aplicação no ITBI Dashboard
✅ Excelente para dark mode — sofisticado e técnico
✅ Surface ladder perfeito para cards de dados
✅ Sem laranja — clean e minimalista
