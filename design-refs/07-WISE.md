# Wise — Design System

## Identidade
- Fundo: sage canvas (#e8ebe6) — "Scandinavian fintech magazine"
- Accent: lime-green (#9fe870) — só para CTAs primários
- Tipografia: Wise Sans weight 900 para display, Inter para body
- Botões: 24px radius (todos), limão no primary
- Cards: 24px radius, sage ou white

## Paleta
- Canvas white: #ffffff
- Canvas sage: #e8ebe6
- Accent lime: #9fe870
- Ink: #0e0f0c
- Deep ink: #163300
- Body: #454745
- Positive: #2ead4b
- Warning: #ffd11a
- Negative: #d03238
- Accent orange: #ffc091
- Accent cyan: #38c8ff

## Tipografia
- Display: Wise Sans 900, 40–126px
- Body: Inter 400/600, 12–20px
- "Peso 900 para hero, 600 para todo o resto"

## Princípios
- Lime-green = exclusivo para CTA primário
- 24px radius em tudo (botões + cards)
- Alternar sage canvas e white card surfaces
- Sem sombras pesadas
- Cores semânticas completas para status
